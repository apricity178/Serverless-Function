# 打卡提醒 - PushPlus 微信推送

## 功能特性

✅ **HTML 网页推送** - 微信收到的是温馨浪漫的网页卡片  
✅ **多 Token 支持** - 同时推送给多个接收者  
✅ **智能时段识别** - 12:00 前自动使用上班提醒，之后使用下班提醒  
✅ **内置温柔提醒词** - 一键填充适合给老婆/女朋友的浪漫提醒  
✅ **纯前端实现** - 无需服务器，配置保存在浏览器本地

## 快速开始

### 1. 获取 PushPlus Token
1. 访问 [pushplus.plus](http://www.pushplus.plus/)
2. 微信扫码登录
3. 复制您的 Token

### 2. 配置应用
1. 用浏览器打开 `index.html`
2. 输入 PushPlus Token（支持多个）
3. 设置提醒时间（如 08:00、18:00）
4. 选择提醒日期（每天/工作日/自定义）
5. 编辑上班/下班提醒内容
6. 点击 **💾 保存配置**
7. 点击 **🧪 测试推送** 验证

### 3. 部署到 GitHub Pages
1. 创建 GitHub 仓库
2. 上传 `index.html`
3. Settings → Pages → Source: main branch
4. 访问 `https://用户名.github.io/仓库名/`

## 推送效果

微信收到的消息是一个**温馨的 HTML 网页卡片**：

- ☀️ **上班提醒**：暖色背景，包含温柔的上班提醒词
- 🌙 **下班提醒**：深色背景，包含浪漫的下班提醒词
- ❤️ **动态爱心**：页面有心跳动画效果
- 📅 **日期时间**：自动显示当前日期和时间


## 注意事项

⚠️ **浏览器定时器限制**：页面关闭后定时器会停止。

### 解决方案
1. **保持浏览器页面常开**（临时方案）
2. **使用 GitHub Actions** 定时调用 PushPlus API
3. **部署到服务器** 使用 Node.js/Python 定时任务

## 自定义

### 修改 HTML 页面样式
编辑 `generateRomanticHTML()` 函数中的 CSS 样式。

### 添加更多提醒词
编辑 `fillMorningContent()` 和 `fillEveningContent()` 函数。

## 技术栈

- 纯 HTML + CSS + JavaScript
- PushPlus API (type=html)
- localStorage 本地存储
